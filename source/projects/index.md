## 🏄🏼划水: 将您的电脑伪装为系统更新

根据你的配置显示一个 Windows Update 画面, 然后你就可以正大光明的划水了.

这个项目是前段时间很火的 UWP 应用[摸鱼](https://www.microsoft.com/zh-cn/p/%E6%91%B8%E9%B1%BC/9ndj3q12nrrm?activetab=pivot:overviewtab)的 Web 版本, 在此基础上解决了一些小问题
